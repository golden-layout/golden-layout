addChild

  • addChild(contentItem: ContentItem, index?: number, suspendResize?: boolean): number

  • Overrides ContentItem.addChild

    • Defined in src/ts/items/row-or-column.ts:125

    Add a new contentItem to the Row or Column

    Parameters

    • contentItem: ContentItem
    • Optional index: number

      The position of the new item within the Row or Column. If no index is provided the item will be added to the end

    • Optional suspendResize: boolean

      If true the items won't be resized. This will leave the item in an inconsistent state and is only intended to be used if multiple children need to be added in one go and resize is called afterwards

    Returns number
